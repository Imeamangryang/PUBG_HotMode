# Death-But-Not-Out 시스템

## 분석

### 기본 규칙

- HP가 0이 되면 즉시 사망하지 않고 **기절(DBNO)** 상태 진입
- 기절 게이지(DBNO HP)가 100%에서 시작, 0%가 되면 사망
- 팀원이 부활시키면 10% HP로 소생
- **솔로 모드에서는 DBNO 없이 즉시 사망**

### 기절 회차별 생존 시간

기절 횟수가 누적될수록 생존 시간이 급격히 짧아집니다.

| 회차   | 생존 시간 |
| ------ | --------- |
| 1회차  | 90초      |
| 2회차  | 30초      |
| 3회차  | 30초      |
| 4회차  | 10초      |
| 5회차  | 7초       |
| 6회차  | 5초       |
| 7회차  | 4초       |
| 8회차+ | 3초       |

> DBNO HP는 항상 100에서 시작하고, **초당 감소 속도**가 회차에 따라 달라지는 방식.
> 1회차: 100 / 90 ≈ 1.11/s, 2회차: 100 / 30 ≈ 3.33/s, ..., 8회차: 100 / 3 ≈ 33.33/s

### 이동 상태에 따른 감소율

- 가만히 있을 때: **더 느린** HP 감소율
- 기어서 이동할 때: 기본 감소율 적용

### 부활

- 팀원이 부활 상호작용 → 부활 완료 시 10% HP로 소생
- 부활 중에도 DBNO HP는 계속 감소

### DBNO 중 피격

- DBNO 상태에서 피격 시 DBNO HP가 직접 감소
- DBNO HP가 0이 되면 사망

## 확인된 사항

| #   | 질문              | 결정                                                                              |
| --- | ----------------- | --------------------------------------------------------------------------------- |
| Q1  | 부활 소요 시간    | 원작 동일 10초 고정, 회차별 변동 없음                                             |
| Q2  | 정지/이동 감소율  | 정지 시 = 기본의 50%. 구체 수치 불명, 50%로 가정                                  |
| Q3  | 기어가기 속도     | 원작 동일 (기본 이동 속도의 약 15%)                                               |
| Q4  | DBNO 중 행동 제한 | 원작 동일. 공격/아이템/인벤토리 불가. 기어가기 + 카메라만 가능                    |
| Q5  | 부활 상호작용     | 원작 동일. 부활자 이동/공격 불가, 범위 이탈/키 떼기/피격 시 취소, 처음부터 재시작 |
| Q6  | 모드 판정         | 추후 GameState에서 모드 정보 제공 예정. 임시로 DBNO 활성화 플래그 사용            |
| Q7  | 동시 부활         | 한 번에 1명만 가능                                                                |
| Q8  | 부활 후 즉시 기절 | 부활 후 0.5초 무적. 이후 기절 시 회차 증가                                        |
